# Constriqo - Borrador de politica de licencia

Este documento no sustituye asesoria legal.

## Uso permitido

El cliente recibe derecho de uso del software en la instalacion contratada.

## Restricciones

- No copiar el codigo fuente.
- No redistribuir el software.
- No revender el software.
- No realizar ingenieria inversa salvo permiso escrito.
- No compartir credenciales administrativas.
- No extraer ni reutilizar componentes para otro producto.

## Datos del cliente

Los datos pertenecen al cliente. Cada instalacion debe mantener base de datos y backups separados.

## Codigo fuente

El codigo fuente permanece propiedad del proveedor salvo acuerdo escrito distinto.
